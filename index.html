    The Patrons Account Information API (PAIA) is a HTTP based programming interface to access library patron information, such as loans, reservations, and fees.

    The preliminary specification of PAIA is managed in a git repository at http://github.com/gbv/paia. Feel free to comment, raise issues and modify the text until it will be released as PAIA 1.0. A snapshot in HTML format is made available from time to time at this page.

    How to contribute

    • Implement PAIA!
    • Formulate the specification!

    The master file paia.md is written in Pandoc Markdown which can be converted to HTML and other formats (see Makefile).
