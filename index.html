Patrons Account Information API (PAIA)

View On GitHub
Specification:
  • current draft

    The Patrons Account Information API (PAIA) is a HTTP based programming interface to access library patron information, such as loans, reservations, and fees.

    The preliminary specification of PAIA is managed in a git repository at http://github.com/gbv/paia.

    How to contribute

    • Implement a server and/or client of the PAIA specification!
    • correct the current PAIA draft!
    • comment on the specification!
    • Think about useful apps and mashups that make use of PAIA!

    Background

    A preliminary version of PAIA has been specified in German as part of a GBV project. This page should help to get a clean and useful specification in English before releasing PAIA 1.0.
